## About This Site

This blog is built using:

- **[Pelican](https://getpelican.com/)** - A static site generator written in Python
- **[Flex Theme](https://github.com/alexandrevicenzi/Flex)** - A beautiful, responsive theme
- **[Cloudflare Pages](https://pages.cloudflare.com/)** - Free, fast hosting with global CDN
- **[Markdown](https://www.markdownguide.org/)** - For writing content

### Why Static Sites?

Static sites are:

1. **Fast** - No database queries, just pre-built HTML
2. **Secure** - No server-side code to exploit
3. **Cheap** - Free hosting on platforms like Cloudflare Pages
4. **Version controlled** - Everything is in Git

Since this is a technical blog, here's a simple Python "Hello World" to test syntax highlighting:

```python
def greet(name):
    """
    A friendly greeting function.

    Args:
        name (str): The name to greet

    Returns:
        str: A personalized greeting
    """
    return f"Hello, {name}! Welcome to Boogerly!"

if __name__ == "__main__":
    message = greet("World")
    print(message)
```
