# Project Overview - Boogerly

**Boogerly** is a personal blog documenting the adventures of Pickem Boogerly in the world of AI-assisted coding and "AI slop" generation. It serves as both a portfolio and a sandbox for experimenting with modern web technologies and AI agents.

## Quick Reference

- **Type:** Monolith Static Site
- **Primary Language:** Python
- **Architecture:** Static Site Generation (SSG)
- **Framework:** Pelican
- **Theme:** Flex

## Key Features

- **Modern UI:** Responsive design with dark mode and syntax highlighting.
- **Privacy First:** Uses Cloudflare Web Analytics instead of intrusive tracking.
- **Automated CI/CD:** Integrated with Cloudflare Pages for seamless deployments.
- **Rich Content:** Support for Markdown with advanced metadata and static asset management.

## Repository Structure Summary

- `/content`: Where the magic happens (Markdown content).
- `/themes`: The visual soul (Jinja2 templates).
- `/pelicanconf.py`: The brain (Site configuration).
- `/tasks.py`: The hands (Automation tasks).
